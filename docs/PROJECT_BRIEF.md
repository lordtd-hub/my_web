# Project Brief

## ชื่อโครงการ

ชื่อชั่วคราว: Personal Academic Learning Portal

ชื่อ public สุดท้ายยังไม่ได้ตัดสินใจ

## วัตถุประสงค์หลัก

สร้างเว็บไซต์ส่วนตัวสำหรับอาจารย์คณิตศาสตร์ระดับมหาวิทยาลัย โดยรวมงาน 4 ส่วนไว้ในโครงการเดียว:

1. เว็บไซต์สาธารณะสำหรับแนะนำตัวตนทางวิชาการ
2. หน้ารายวิชาและสื่อสำหรับการเรียนรู้ด้วยตนเอง
3. ระบบนักศึกษาสำหรับดูคะแนนส่วนตัวอย่างปลอดภัย
4. แดชบอร์ดอาจารย์สำหรับจัดการรายวิชา นักศึกษา และคะแนน

## ภาษาและน้ำเสียง

ใช้ภาษาไทยเป็นหลักสำหรับข้อความที่ผู้ใช้เห็นและเอกสารโครงการ

public website ควรใช้ภาษาไทยเชิงวิชาการที่สุภาพ กระชับ อ่านง่าย และเหมาะกับเว็บไซต์ของอาจารย์คณิตศาสตร์ระดับมหาวิทยาลัย ไม่ควรใช้ถ้อยคำเชิงโฆษณามากเกินไป

ข้อความสำหรับนักศึกษาควรสุภาพ ชัดเจน และเข้าใจง่าย ข้อความสำหรับอาจารย์หรือ admin ควรตรงประเด็น ใช้งานได้จริง และไม่เยิ่นเย้อ

คำศัพท์ทางเทคนิค เช่น Firebase, Firestore, Authentication, Hosting, Security Rules, CSV, UID, Vite, React และ TypeScript ควรคงเป็น English เมื่อช่วยให้เข้าใจชัดกว่า

ห้ามแต่งข้อมูลจริงของอาจารย์ มหาวิทยาลัย รายวิชา นักศึกษา หรือคะแนน ให้ใช้ placeholder จนกว่าจะได้รับข้อมูลจริงจากผู้ใช้

## กลุ่มผู้ใช้

### ผู้เข้าชมทั่วไป

อาจประกอบด้วย:

- นักศึกษา
- เพื่อนร่วมงาน
- นักวิจัย
- บุคลากรของมหาวิทยาลัย
- ผู้ที่ค้นหาชื่ออาจารย์ผ่านระบบค้นหา

ควรสามารถดูข้อมูลต่อไปนี้ได้:

- ประวัติทางวิชาการ
- ข้อมูลการสอน
- ความสนใจด้านวิจัย
- โครงการหรือผลงาน
- คำอธิบายรายวิชาที่เปิดเผยได้
- ช่องทางติดต่อที่อาจารย์อนุญาตให้เผยแพร่

### นักศึกษาที่ลงทะเบียนเรียน

ควรสามารถ:

- เข้าสู่ระบบ
- ดูรายวิชาที่ตนลงทะเบียน
- ดูสื่อการเรียนรู้ที่เกี่ยวข้องกับรายวิชา
- ดูคะแนนของตนเอง
- ดู feedback จากอาจารย์
- ดูประกาศของรายวิชา

### อาจารย์หรือ admin

ควรสามารถ:

- จัดการเนื้อหา public ที่เกี่ยวข้องกับรายวิชา
- จัดการรายวิชา
- จัดการ enrollment
- import คะแนนจาก CSV
- publish หรือ unpublish score items
- ดูภาพรวมคะแนนของรายวิชา

## ขอบเขต MVP

เวอร์ชันแรกควรมี:

- public website
- หน้ารายวิชา
- Firebase login
- ระบบนักศึกษา
- หน้าคะแนนนักศึกษา
- แดชบอร์ดอาจารย์
- CSV score import
- Firestore Security Rules

## สิ่งที่อยู่นอกขอบเขต MVP

ห้ามเพิ่มสิ่งต่อไปนี้ เว้นแต่ผู้ใช้สั่งชัดเจน:

- online quiz engine
- assignment submission
- auto grading
- Cloud Functions
- Cloud Storage
- payment features
- AI tutor
- discussion forum
- video hosting
- production deployment
